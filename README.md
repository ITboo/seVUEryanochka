# SeVUEryanochka

Repository for the study project "Severyanochka". It's made for the code practice with Vue, to get to know how to develop such a kind of projects and how to work with a big enough Figma layouts.

The project is an online grocery-store. It has an adaptive layout and a mobile application.

## Stack

Vue.js (Vue3), TypeScript... *to be continued*
